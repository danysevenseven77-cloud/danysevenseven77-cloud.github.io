About Combat Compass

Combat Compass is a martial arts website designed to help people learn about different fighting styles and find the martial art that fits them best. The website includes information about popular martial arts, their benefits, and a quiz that recommends a style based on the user's goals and preferences.

This project was created using HTML, CSS, Bootstrap 4.6, and JavaScript. Bootstrap was used to help with the layout and responsive design, while JavaScript was used to make the quiz interactive.

The goal of this website is to make martial arts easier to understand for beginners who may not know where to start.

Technologies Used

  • HTML for the page structure
  • CSS for colors and styling
  • Bootstrap 4.6 for layout and responsive design
  • JavaScript for the martial arts recommendation quiz
